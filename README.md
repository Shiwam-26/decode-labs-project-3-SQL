# 🗄️ SQL Data Analysis Project

> **Decode Labs Internship | Week 3 Project**

## 📌 Project Overview

This project demonstrates practical SQL skills by solving real-world business problems using a relational database. During this project, I performed data exploration, cleaning, aggregation, filtering, joins, subqueries, and analytical queries to extract meaningful insights from the dataset.

The objective was to strengthen SQL proficiency while answering business-oriented questions through efficient and optimized SQL queries.

## 🎯 Objectives

* Understand the database schema
* Explore the dataset using SQL
* Perform data cleaning
* Write business-driven SQL queries
* Analyze customer and sales data
* Generate actionable insights
* Practice advanced SQL concepts
